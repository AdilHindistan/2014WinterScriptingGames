## Security Plugin Configuration file
## <configuration_item>:$true|$false
## A value of $true means script should process the relevant plugin code to get data on the item
 
Get-SecEventLog:$true
Get-SecFolder:$true
Get-SecFile:$true
Get-SecShare:$true
Get-SecInstalledSoftware:$true
Get-SecProcess:$true
Get-SecService:$true
Get-SecEnvVariable:$true
Get-SecRegistry:$true
